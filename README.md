## Getting started

Install the package in your project directory with:

```bash
// with npm
npm install @commitlint/cli @0x219/commitlint-config --save-dev

// with yarn
yarn add @commitlint/cli @0x219/commitlint-config --save-dev
```

In commitlint.config.js:

```js
module.exports = {
  extends: ['@0x219'],
};
```
